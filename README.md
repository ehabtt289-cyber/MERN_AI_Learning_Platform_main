# MERN AI Learning Platform

An AI-powered learning platform built using the MERN stack that helps users improve learning through quizzes, flashcards, document processing, and progress tracking.

---

# Features

- AI Learning System
- Quiz Generation
- Flashcards Management
- Progress Tracking
- Document Upload & Processing
- Authentication System
- REST API Integration
- Responsive User Interface

---

# Tech Stack

## Frontend
- React.js
- Vite
- Axios
- CSS

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

---

# Project Structure

frontend/
backend/

---

# Installation

## Clone Repository

git clone https://github.com/ehabtt289-cyber/MERN_AI_Learning_Platform_main.git
cd MERN_AI_Learning_Platform_main

---

# Backend Setup

cd backend
npm install

Create .env file inside backend folder:

env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
GEMINI_API_KEY=your_gemini_key
ALLOWED_ORIGINS=yourn_url_frontend

Run backend server:

npm run dev

---

# Frontend Setup

cd frontend
npm install
npm run dev

---

# Environment Variables

## Backend .env

env
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret

## Frontend .env

env
VITE_API_URL=http://localhost:5000
